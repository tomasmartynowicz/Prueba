COMANDOS PARA GIT: git config --global user.name "tu nombre".
		   git config --global user.email "<tu_email>".
		   cd selecciona el repositorio de trabajo.
		   mkdir crea una carpeta dentro del repositorio.
		   git clone http://github.com/tomasmartynowicz/prueba copia
		   el repositorio web.
		   touch archivo.extencion crea un archivo vacio.
		   vi archivo.extencion abre el editor de texto del SO.
		   git add.extencion agrega el archivo determinado al 
		   repositorio.
		   git add . agrega todos los archivos creados al repositorio.
		   git status muestra los archivos agregados.
		   git commit abre primero el editor del commit para luego 
		   guardarlo en el repositorio.
		   git push   mueve los archivos modificados al repositorio
		   :wq cierra el editor y guarda en el archivo.
